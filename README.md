# Instagram-Engagement-Rate-Using-Machine-Learning.

## Overview
This project predicts Instagram Engagement Rate using Machine Learning techniques. Multiple regression models were trained and compared to determine the most accurate prediction model.

## Features
- Data Cleaning & Preprocessing
- Exploratory Data Analysis (EDA)
- Feature Encoding
- Model Training & Evaluation
- Engagement Rate Prediction

## Technologies Used
- Python
- Pandas
- NumPy
- Scikit-Learn
- Matplotlib
- Seaborn
- Joblib

## Models Implemented
- Linear Regression
- K-Nearest Neighbors (KNN)
- Random Forest Regressor
- Decision Tree Regressor

## Best Model
Decision Tree Regressor achieved the highest performance and was selected as the final model.

## Evaluation Metrics
- MAE
- MSE
- RMSE
- R² Score

## Project Structure
├── predict_instagram_engagement_rate.ipynb
├── Batch C2 Instagram_Analytics.csv
├── trained_models/
├── label_encoders.pkl
└── README.md

## Advantages
- Supports data-driven content decisions
- Predicts engagement before publishing
- Easy-to-interpret model results
- Compares multiple ML algorithms

## Limitations
- Depends on dataset quality
- May not generalize to future Instagram trends
- External factors are not considered
- Decision Trees can overfit
